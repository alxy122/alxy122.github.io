MERKT - Mensch-Reaktions-und-Korrelations-Test


Willkommen auf unserer Website. Hier finden Sie Links zu den drei Tests. Am Ende jedes Tests erscheint ein Download-Button mit den Daten aus dem abgeschlossenen Test. Diese Datei bitte im ILIAS-Forum hochladen, damit wir sie auswerten können.

Test 1: Reaktionstest mit erscheinenden Kreisen
Test 2: Kreise und Quadrate unterscheiden
Test 3: Länder in ihren Kontinent einordnen
Bonus: Highscore-Jagd!

Jeder Test läuft für 30 Eingaben. Dabei zählt nur Test 1 auch Eingaben, die getätigt werden, wenn kein Stimulus zu sehen ist (was als Fehler gewertet wird). Vorzeitig können die Tests durch Drücken von [a] beendet werden. Das Bonus-Spiel ist eine Abwandlung von Test 2, in der sich die Geschwindigkeit stetig erhöht.
